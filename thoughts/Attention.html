Attention operates on a sequence of query Q, key K and value V vector. Attention matrix of a sequence then computed as:

A(Q,K,V)=softmax(d​Q⋅KT​)V   for QL×d​,KL×d​,VL×d​

Paged Attention

paper

Used in conjunction with Continuous batching

Reduce memory usage of attention mechanism by swapping kv-cache in and out of memory. A block manager is similar to those of virtual memory in OS.

Essentially, it’s a form of paging, such that attention can be stored in contiguous memory. Partitions the KV cache of each sequence into KV blocks.

Given:

  • each block contains KV vectors for fixed number of tokens, denoted as block size B.
  • Key block Kj​=(k(j−1)B+1​,…,kjB​)
  • Value block Vj​=(v(j−1)B+1​,…,vjB​)
Aij​=∑t=1i//B​exp(qiT​Kt​/d​)exp(qiT​Kj​/d​)​,oi​=j=1∑i//B​Vj​AijT​

where Aij​=(ai,(j−1)B+1​,…ai,jB​) is row vector of attention score on j-th KV block.
